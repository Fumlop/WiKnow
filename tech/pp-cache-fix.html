PP Cache Fix

How to fix a broken PowerPlay data cache in Elite: Dangerous.

Before starting: Close Elite: Dangerous and the Elite launcher completely.

The Fix

Rename (or delete) the file:

C:\Users\<username>\AppData\Local\Frontier Developments\Elite Dangerous\GalacticPoliticsPowers2Digest.cache

(where <username> is your Windows username)

Step by Step

The AppData folder is normally hidden, so the easiest way to get there:

  1. Press Win+R at the Windows desktop to open the Run window
  2. Copy and paste %LOCALAPPDATA%\Frontier Developments\Elite Dangerous into the Open text box
  3. Click OK
  4. Find the GalacticPoliticsPowers2Digest.cache file
  5. Rename or delete it
  6. Launch Elite and test
Choose carefully: There are two files with similar names in that folder. Make sure you pick GalacticPoliticsPowers2Digest.cache (with the 2).
